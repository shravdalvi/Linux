What is ls?

List Directories

$ ls

$ ls /home/pete

Also note that not all files in a directory will be visible. 
Filenames that start with . are hidden, you can view them however with the ls command and pass the -a flag to it (a for all).

$ ls -a

s.Dalvi@LT0221484 MINGW64 ~/Desktop/shrav
$ ls -a
 ./          'Application Number License.pdf'  'Exp 2 CODE DS.docx'    'Git Bash.lnk'*           Passport/
 ../         'CAP ROUNDS'/                     'EXPERIMENT NO 1.docx'   Git-2.47.1-64-bit.exe*   Sem/
 Admission/  'E-Aadhar Card.pdf'               'Experiment No 2.docx'   License/
 all/        'Exp 1 CODE DS.docx'              'EXPERIMENT NO 3.docx'   PAN/

There is also one more useful ls flag, -l for long, this shows a detailed list of files in a long format. 
This will show you detailed information, starting from the left: file permissions, number of links, owner name, owner group, file size, timestamp of last modification, and file/directory name.

s.Dalvi@LT0221484 MINGW64 ~/Desktop/shrav
$ ls -l

total 80

drwxr-x--- 7 pete penguingroup   4096 Nov 20 16:37 Desktop
